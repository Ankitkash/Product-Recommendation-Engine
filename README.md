# Product-Recommendation-Engine


A machine learning based product recommendation system built with **Python** and **Streamlit**.
Recommends similar products using **KNN**, **KMeans**, **Popularity Scoring**, and a **Hybrid** model.

## Project Overview

This project builds a product recommendation system using machine learning.  
It takes a product name as input and recommends **5 similar products** using 4 different methods.  
The app runs directly in the browser using Streamlit.

---

## Features

- Search any product by name or keyword
- 4 recommendation methods — KNN, KMeans, Popularity, Hybrid
- Displays product image, title, rating, price and reviews
- Fallback placeholder image if product image is unavailable
- Pre-trained models loaded from `.pkl` file for fast startup
- Clean and simple browser-based UI

## How It Works

```
CSV file
   ↓
load_data()       →   clean, deduplicate, sample 50k rows
   ↓
MinMaxScaler      →   scale stars, reviews, price, boughtInLastMonth to [0, 1]
   ↓
KNN model         →   find 5 nearest neighbours by Euclidean distance
KMeans model      →   group all products into 10 clusters
Popularity score  →   stars×0.4 + reviews×0.35 + boughtInLastMonth×0.25
   ↓
models.pkl        →   save everything
   ↓
recommender.py    →   load models.pkl → user searches → results displayed
```

## Models Explained

### 1. KNN — K-Nearest Neighbours

Finds the 5 most similar products based on **Euclidean distance** across all 4 scaled features.  
Products that are similar in rating, price, reviews, and purchase count appear close together and get recommended.

### 2. KMeans Clustering

Groups all products into **10 clusters** based on feature similarity.  
When you search a product, it finds which cluster that product belongs to and returns the top 5 highest rated products from the same cluster.

### 3. Popularity Score

No search needed. Ranks all products by a weighted formula:

```
popularity_score = (stars × 0.4) + (reviews × 0.35) + (boughtInLastMonth × 0.25)
```

> All values are MinMax scaled before scoring so no single column dominates.

### 4. Hybrid — Best Method

Combines KNN similarity and popularity score with a **70/30 blend**:

```
hybrid_score = (0.7 × similarity_score) + (0.3 × popularity_score)
```

> Similarity score = `1 / (1 + distance)` — keeps value between 0 and 1.  
> Products that are both **similar AND popular** rank highest.

## How to Run

```bash
streamlit run recommender.py
```

App opens automatically at:

```
http://localhost:8501
```

To stop the app:

```bash
Ctrl + C
```

---

## How to Use the App

1. Open `http://localhost:8501` in your browser
2. Select a recommendation method from the dropdown

| Method | Description |
|---|---|
| **Hybrid** | Best overall — blends similarity + popularity |
| **KNN** | Purely feature similarity based |
| **KMeans** | Cluster based recommendations |
| **Popularity** | No search needed — shows trending products |

3. Type a product name or keyword — e.g. `wireless headphones`, `laptop`, `face cream`
4. Click **Recommend**
5. See 5 product cards with image, rating, price and reviews
